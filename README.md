### **Prerequisites Checklist**
- ‚úÖ **Node.js 18+** with npm/yarn package manager
- ‚úÖ **Modern Web3 Wallet** (MetaMask, WalletConnect, Coinbase Wallet)
- ‚úÖ **Testnet ETH** for gas fees (Sepolia testnet recommended)
- ‚úÖ **Basic DAO Knowledge** (governance concepts and voting mechanisms)

### **Installation Steps**

#### 1. **Clone the Repository**
```bash
git clone https://github.com/cameronking88/shield-vote.git
cd shield-vote
```

#### 2. **Install Dependencies**
```bash
npm install
# Alternative: yarn install
```

#### 3. **Environment Setup**
```bash
cp env.example .env.local
```

**Configure Environment Variables:**
```env
# Network Configuration
VITE_CHAIN_ID=11155111
VITE_RPC_URL=your_sepolia_rpc_endpoint_here

# Wallet Integration
VITE_WALLET_CONNECT_PROJECT_ID=your_walletconnect_project_id_here

# Smart Contract Addresses
VITE_SHIELD_VOTE_CONTRACT_ADDRESS=your_deployed_contract_address_here

# FHE Network Configuration
VITE_FHE_NETWORK_URL=your_fhe_network_endpoint_here
```

#### 4. **Launch Development Server**
```bash
npm run dev
# Alternative: yarn dev
```

**üéâ Success!** Visit `http://localhost:5173` to experience Shield Vote!
